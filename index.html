Research

This project combines web development with an evaluation of ChatGPT’s performance across various domains. By creating a user-friendly website, it documents research findings, showcases data visualizations, and highlights insights into ChatGPT’s efficiency and versatility. The goal is to bridge technical analysis with practical application, providing an engaging platform that demonstrates the potential of AI while developing essential skills in web design, data handling, and API integration.

Research Approach

We used tools and technologies such as MongoDB, Node.js, and the ChatGPT API to evaluate ChatGPT’s performance across domains.

The dataset consists of questions across History, Social Science, and Computer Security. This evaluation aims to measure ChatGPT’s accuracy, versatility, and efficiency in different areas.
